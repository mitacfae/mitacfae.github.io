How to Remotely Upgrade R2520G6 Firmware Using Redfish Commands

Date: 2026-06-11 | Last Updated: 2026-06-11

Tags: R2520G6 Redfish Firmware BIOS BMC CPLD ROT

Description

When performing server maintenance, manual or onsite firmware updates often introduce numerous challenges and inefficiencies. To improve operational efficiency and implement robust remote management, an automated solution is required to streamline the firmware update process for the R2520G6 server.

Solution

This solution provides a comprehensive guide on how to utilize Redfish API commands to remotely upgrade the BIOS, BMC, CPLD, and ROT firmware of the R2520G6 server[2, 10].

Prerequisites

Before executing any Redfish update commands, ensure that you complete the following preparatory steps:

  1. Power Off the Server: Ensure that the R2520G6 is completely powered off before executing any Redfish upgrade commands.
  2. Prepare Firmware Files: Upload the required out-of-band (OOB) firmware upgrade image files to your Linux working directory.

Detailed Firmware Upgrade Steps

1. BIOS Firmware Upgrade Process

  • Step 1: Check Current Version
    Verify the currently active BIOS version using the following Redfish command: 
    curl -u username:password -k -s https://{BMC_IP}/redfish/v1/UpdateService/FirmwareInventory/bios_active
  • Step 2: Define Update Target
    Use the PATCH command to define the update target as bios_active: 
    curl -k -u username:password -X PATCH https://{BMC_IP}/redfish/v1/UpdateService -H "Content-Type: application/json" -d '{"HttpPushUriOptions": {"HttpPushUriApplyTime": {"ApplyTime": "Immediate"}},"HttpPushUriTargets": ["bios_active"],"HttpPushUriTargetsBusy": true,"Oem": {"ApplyOptions": {"ClearConfig": true}}}' -w "\nHTTP_CODE=%{response_code}\n"
    (Expected Response HTTP Code: 200)
  • Step 3: Execute Firmware Upgrade
    Use the POST command to upload the corresponding upgrade image file: 
    curl -k -u username:password -X POST https://{BMC_IP}/redfish/v1/UpdateService/update -H "Content-Type: application/octet-stream" --data-binary "@{Image_path}" -w "\nHTTP_CODE=%{response_code}\n"
    (Expected Response HTTP Code: 202)
  • Step 4: Wait and Verify
    A new task will appear in the BMC Task menu[29]. Please wait approximately 10 minutes for the BIOS upgrade process to complete[29]. Once finished, check the firmware version again to verify success.
