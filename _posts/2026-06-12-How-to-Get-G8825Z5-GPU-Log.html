Question

How to get G8825Z5 GPU log?

Requirement

  • Hardware: G8825Z5 Server
  • CPU: AMD EPYC 9575F x2
  • RAM: Samsung DDR5 5600 RDIMM (M321R8GA0PB0-CWMXJ)
  • OS: Ubuntu 22.04 Server
  • Application: UBB_allLog_collection.sh.x

Operation

Follow the steps below to collect GPU logs from the G8825Z5 server. 


# Step 1: Copy log collection tool
cp UBB_allLog_collection.sh.x ~

# Step 2: Change permission
chmod +x UBB_allLog_collection.sh.x

# Step 3: Execute script
sudo ./UBB_allLog_collection.sh.x

Result

After execution is completed, the following log package will be generated: 


All_logs.tar.gz

Please send the generated All_logs.tar.gz file to MiTAC for further analysis.
